# Sample Data

本目录保存接手文档使用的脱敏样例数据。样例来自当前本地运行产物，只保留少量行用于说明字段结构，不是完整数据集，也不构成投资建议。

文件清单：

- `ranking_latest_sample.csv`：当前首页每日预测排名样例。
- `historical_ranking_sample.csv`：历史回放使用的每日 ranking 样例。
- `final_recommendations_sample.csv`：AI 调整后的最终推荐样例。
- `paper_account_latest_sample.json`：模拟盘账户样例。
- `paper_positions_latest_sample.csv`：模拟盘持仓样例。
- `paper_orders_sample.csv`：模拟盘历史订单样例。
- `ai_paper_decisions_latest_sample.json`：模拟盘决策样例。
- `news_stock_mapping_sample.csv`：新闻和股票映射样例。
- `news_event_sample.csv`：新闻事件样例。

字段解释见 `../handoff/DATA_DICTIONARY.md`。
